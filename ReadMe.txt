PrepiApp:
this is the frontend which is built on expo and react native

Express_backend:
this is the backend which is built on express and uses node.js

git clone
cd myexpoapp
npm install
npx expo start


Frontend
- Expo
- React Native
- JavaScript

Backend
- Node.js
- Express

Get these to run
- Node.js
- npm
- Expo Go

Frontend start-up:
cd PrepiApp
npm install
npx expo start

Backend start-up:
cd Express_backend
npm install
npm start


Before running frontend:
Edit line 28 of PrepiApp/app/(auth)/index.js to contain the IP address of the machine you're running the backend from. Do the same on line 6 of recipe.js too.
